Privacy Policy

FolderLM — Workspace Organizer for NotebookLM
Last updated: March 27, 2026

Overview

FolderLM is a Chrome extension that helps you organize your Google NotebookLM notebooks into folders. We are committed to protecting your privacy. This policy explains what data we collect, how we use it, and your rights.

Data Collection

FolderLM does not collect, transmit, or share any personal data.

All data created by FolderLM — including folder names, notebook assignments, and your preferences — is stored locally on your device using the Chrome Storage API. No data is ever sent to external servers.

What We Store Locally

  • Folder names and hierarchy you create
  • Notebook-to-folder assignments
  • UI preferences (sidebar width, theme)

This data never leaves your browser. It is synced only if you have Chrome Sync enabled for extension data, which is managed by Google, not by FolderLM.

Permissions

  • storage — Used to save your folder structure and preferences locally.
  • sidePanel — Used to provide the side panel interface.
  • Host permission (notebooklm.google.com) — Required to inject the sidebar UI into the NotebookLM page. FolderLM only runs on this specific site.

Third-Party Services

FolderLM does not use any third-party analytics, tracking, or advertising services.

Data Export & Deletion

You can export your workspace data at any time using the built-in export feature. To delete all data, simply uninstall the extension — all locally stored data will be removed automatically.

Changes to This Policy

If we update this policy, the changes will be posted on this page with a revised date.

Contact

If you have questions about this privacy policy, please open an issue on our GitHub repository.
